The biologically inspired activation function.

## Usage

Class version

```python
from torch import nn
from sigmoidcolon import SigmoidColon

model = nn.Sequential(
    nn.Linear(784, 64),
    SigmoidColon()
    nn.Linear(64, 10)
)
```

Functional version

```python
from torch import nn
from sigmoidcolon.functional import sigmoidcolon

x = torch.randn(100)
y = sigmoidcolon(x)
```
